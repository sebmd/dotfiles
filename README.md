# Instalacja

```
curl https://git.io/JULqp -fLo $HOME/git/get-repo.sh --create-dirs
cd $HOME/git
bash ./get-repo.sh
```

**Uruchomienie instalatora**

```
cd $HOME/git/dotfiles
./install.sh
```
