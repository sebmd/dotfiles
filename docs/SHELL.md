# Polecenie fd

```
fd . --ignore-file .gitignore --hidden
```

Ignorowanie plików zawierających się w podanym pliku np. `.gitignore`

```
--ignore-file .gitignore
```

Wyszukiwanie ukrytych plików zaczynających się od znaku `.`
```
--hidden
```
